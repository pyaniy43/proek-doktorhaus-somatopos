🚀 ШАГ 2: БЫСТРЫЙ ЗАПУСК

Всё стало намного проще!

1. Распакуй скачанную ZIP-папку проекта в удобное место
2. Зайди в папку и найди файл "start.bat" (с шестеренками/черным значком)
3. Дважды кликни по "start.bat"

Скрипт автоматически:
- Скачает все зависимости и библиотеки.
- Обучит нейронную сеть.
- Сам откроет три нужных окна для сайта, нейросети и пульта симуляции!

❓ ЧТО ЗА ОКНА И ЗАЧЕМ ИХ 3?

1. Окно "САЙТ" — это веб-сервер, который держит твой сайт (localhost:3000).
2. Окно "НЕЙРОСЕТЬ" — это фоновая программа, которая непрерывно «слушает» входящие медицинские данные, обрабатывает их и выдает диагноз.
3. Окно "ПУЛЬТ" — тестовый скрипт, через который ты можешь симулировать подключение прибора и вручную нажимать «сценарий 1», «сценарий 2» и отправлять фейковые показатели для проверки.

В реальной эксплуатации окно №3 заменится реальным медицинским оборудованием.
